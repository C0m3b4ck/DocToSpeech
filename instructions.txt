1. Install python3.11 - 'sudo apt install python3.11 python3.11-venv' on Ubuntu
2. Make venv - 'python3.11 -m venv yourvenvname'
3. Activate venv - 'source yourvenvname/bin/activate'
4. Upgrade pip - 'pip install --upgrade pip'
5. Install requirements.txt - 'pip install requirements.txt'
6. Run doctospeech.py - 'python doctotts.py'
